# Fuck-CSU-Run

基于 JavaScript 实现的CSU课外体育 API 封装库。

### 声明

* 项目作者不对使用本项目造成的任何后果承担任何责任

### 功能

* **用户信息获取**：获取个人跑步进度、历史记录及学期统计。
* **模拟跑步流程**：完整覆盖 `开始跑步 -> 坐标打卡 -> 轨迹上传 -> 结束跑步` 的全生命周期。
* **多校区支持**：支持南校、湘雅、本部、新校、铁道校区的打卡点。

### 接口流程图

* 懒得画，按`开始跑步 -> 坐标打卡 -> 轨迹上传 -> 结束跑步`的顺序即可

### 注意事项

1. **global.dataKey**： `global.dataKey` (签名密钥)可能会随版本更新改变。截至本项目发布，该密钥可以正常使用。
2. **user.brand/user.model**： 除非你是第一次开始跑步，否则校园跑服务器会记录你的常用设备型号。型号一致才能开始跑步。
3. **wx.login**：结束跑步时需要使用wx.login返回的一次性登录码，建议抓包/login拦截使用。
4. **轨迹文件**：`uploadFile` 接口需上传符合格式的经纬度坐标点数组。经纬度可以不在电子围栏范围内，甚至可以不上传轨迹。结束跑步后就算分了。
